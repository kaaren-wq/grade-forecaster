# TMU Grade Calculator

A Streamlit app built specifically for Toronto Metropolitan University students.

## Features
- TMU 4.33 GPA conversion
- TMU letter grade mapping
- Standard rounding before conversion
- Final exam slider simulator
- Multiple charts
- Streamlit duplicate-chart bug fixed with unique keys

## Run
```bash
pip3 install streamlit pandas numpy plotly
python3 -m streamlit run tmu_grade_calculator.py
```
